## Структура БД

```
Users             — пользователи (id, username, email, passwordHash, role, createdAt)
Tickets           — тикеты (id, number, title, description, status, priority, ...)
Comments          — комментарии (id, ticketId, authorId, text, isInternal, createdAt)
HistoryEntries    — история изменений (id, ticketId, field, oldValue, newValue, comment, createdAt)
Categories        — категории (id, name)
SlaConfigs        — нормативы SLA (id, priority, reactionHours, resolutionHours)
TicketFiles       — файлы (id, ticketId, filename, path, createdAt)
TicketDependencies — зависимости FR-23 (ticketId, blockedById)
AuditLogs         — журнал действий ОБ-7 (id, userId, action, entityType, entityId, createdAt)
```
